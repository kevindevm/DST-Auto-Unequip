Client mod. Automatically unequips your rechargable items at 1% to 10% for normal items, exclusive 1% to 60% to shield of terror, eye mask and Dreadstone set

[hr][/hr] 

[h1] Configuration [/h1]

[b]* Normal items to unequip at 1 to 10% > [/b] [u]default set to 5% [/u]
[b]* Shield of terror and eye mask to unequip from 1% to 60% >[/b] [u]default set to 25% [/u]
[b]* Dreadstone SET 1% to 60% >[/b] [u]default set to 5% [/u]
[b]* Ignore hand slot items >[/b] [u]default set to YES [/u][b](set to false to work with Shield of terror)[/b]
[b]* Force retrying unequip >[/b] [u]default set to true[/u]
[b]* Include Bee Queen Crown>[/b] [u]default set to false [/u]
[b]* Include Commander's Helm>[/b] [u]default set to false [/u]
